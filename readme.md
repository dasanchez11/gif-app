# Gif App

1. Clone the repository

2. Create a **.env**

3. Add your https://giphy.com/ api key to the .env

```
VITE_API_KEY= YOUR KEY;
```

4. Run the container\*\*

```
docker compose up -d

```

5. \*\*Be sure you have a the docker Desktop installed
